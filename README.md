## 🌟 Architecture & Design System

This portfolio and its enclosed applications have been meticulously engineered from the ground up to reflect a **"FAANG-level" standard of quality**:
- **Modern Glassmorphism**: Leveraging backdrop filters and semi-transparent layers for a premium depth effect.
- **Custom Neumorphism**: Integrating soft, extruded UI elements (e.g., in the Calculator app) for tactile feedback.
- **Optimized Performance**: Vanilla JavaScript and CSS variables ensure zero dependencies and blazing-fast load times.
- **Interactive Micro-animations**: State-driven UI changes utilizing `IntersectionObserver` and CSS keyframes for fluid transitions.

## ⚙️ Local Setup

To run this portfolio locally without deploying:

1. **Clone the repository**
   ```bash
   git clone https://github.com/amit2003-cse/amit-portfolio.github.io.git
   cd amit-portfolio.github.io
   ```
2. **Serve the project**
   Since this leverages modern ES6 and fetch (in some extensions), running it via a local development server is recommended:
   ```bash
   npx serve .
   ```
   *Alternatively, use the VSCode "Live Server" extension.*
3. **Open in Browser**
   Navigate to `http://localhost:3000`
